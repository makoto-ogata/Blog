初めてのアメリカ旅行その5

Sep 14, 2017 #travel #America

サンフランシスコも最終日。
翌日にはポートランドへ行く予定。

一応、IT業界の末端で飯食ってる人間としてはせっかくサンフランシスコに来たからにはシリコンバレーは行っておくべきだろうなと思い、午前中から出かけました。

カルトレインに乗ってサンノゼへ

ホテルからmuniメトロに乗ってカルトレインの始発駅のあるサンフランシスコ4番＆キングストリート駅へ。

地球の歩き方「サンフランシスコとシリコンバレー」によると

シリコンバレーは市の集合体で、サンフランシスコの南約50kmに位置するサンタクララカウンティ一帯の事を指す。
一説によると「マイクロ・エレクトロニクス」誌の編集者だったドン・ヘフラーが1971年にハイテク産業が進出してきたサンノゼ周辺を、シリコンバレーと呼んがのが始まりといわれている。

中略

サンタクララカウンティ全体を網羅するVTAバスやライトレイルもあり、路線によってはとても便利。ただし、効率よく動くなら車は必要不可欠。

とのこと。
バスや路面電車があれば何とか回れるだろと安直な考えでサンノゼ行きの切符を購入。

カルトレインはゾーン（区間）毎に料金が違い、サンノゼはゾーン4に位置してます。 ゾーン4でサンフランシスコ市内から往復で約20ドル。

カルトレインに乗ってシリコンバレーへ。

Makoto Ogataさん(@makoto_ogata)がシェアした投稿 - 2017 7月 3 9:38午前 PDT

カルトレインの車内は2階建てになってて、カッコいいです。

そして1時間半ほどでサンノゼ駅に到着。

外は真夏日！
気温が17度くらいしかなかったサンフランシスコから電車で1時間半も行けば気温が全然違うって面白い国だなー。

到着したは良いが、土地勘が全く無いので肝心の行ってみたい有名企業の場所が分からない。
こんな時は迷わずGoogle Mapを開き検索。
とりあえずGoogle本社を検索。

え…公共の乗物で1時間かかるの…。
しかもサンフランシスコ方面に戻るのか…（知ってたら途中下車したのに）

じゃあFacebook本社はどこかな…。

さらに遠い…！！
しかも公共の交通機関じゃなくて、車持ってる前提！

じゃあapple本社は？？？

あ、ここから1番近い。
でも公共の交通機関使ったらやっぱり1時間かかるのか。
「サンノゼ周辺をシリコンバレー」と書いてあったので、サンノゼ駅がハブになる駅だと勝手に思い込んでサンノゼまで来たら、そんなことはなかった。
タクシーとか拾って回れば良いけど、肝心のタクシーが無い。
多分、ここの人達はUberとかLyftとかを使っているのでしょう…。

とりあえずappleしか行けないかなと思い、目的地をapple本社に決めてバスを乗り継いで行こうとしたところ旅行用に買ったSIMカードのデータ通信が底をつく（ついてない！）

持ってるiPhoneが完全なオフライン使用に成り下がったので、仕方なく徒歩圏内で行ける場所を観光する事にしました（データローミングでやろうと思えばできたけどやらなかった）

美術館に行ったり

公園でのんびりしたり

公園のノマドワーカーをぼんやり眺めたりしてました（何しに来たんだ…）

一通り徒歩で回れるところは行ったのでサンフランシスコ市内に戻ろうと思って、帰りの電車に乗ったところ、スパニッシュ系のカップルが僕に「この電車ってサンフランシスコ行くよね？」と聞いてきたので「多分ね」と答えたら「多分て（笑）」と苦笑されました。
すまんな、何もかも初めてなんで分からんのだよ。

サンフランシスコ市内に到着して、疲れてたけどまだ時間はあったのでサンフランシスコ現代美術館に行き、ちょうどムンク展が開催されてたのでムンクの作品みたりミュージアムショップでウィンドウショッピングしたりしてサンフランシスコの旅は終了。

明日はポートランドに向かいます。

つづく。

Makoto Ogata
